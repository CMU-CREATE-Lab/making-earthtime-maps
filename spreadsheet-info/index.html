How to make an Earthtime map:

Link to Google spreadsheet: some link

Columns

  1. Enabled: TRUE/FALSE to show up on map
  2. Share link identifier: hopefully unique
  3. Category: Category in Data Library browse
  4. Name Name that shows up in Data Library browse
  5. Credits
  6. Base Layer: Background map
  7. Custom slider ticks: not working
  8. Start Date:
  9. EndDate:
  10. Step: Ticks on slider essentially: hours/days/years/etc.
  11. URL(see below)
  12. Scaling:
  13. Color Scaling:
  14. Map Type: Bubble, point, chloropletch, etc.
  15. Color: Default color for all drawn elements(mostly for bubble map)
  16. External GeoJSON:
  17. Name Key:
  18. Show Graph: Can show graph of data with map
  19. Graph Title
  20. Graph Plot Colors
  21. Graph Plots
  22. Graph Plots First Visible
  23. Graph X-Axis Label
  24. Graph Y-Axis Label
  25. Graph Y-Axis Min
  26. Graph X-Axis Label Interval
  27. Legend Content: Custom SVG of legend
  28. Legend Key: Add extra info text to legend
  29. Load Data Function
  30. Set Data Function
  31. Number of Levels:
  32. Number of Attributes: attributes taken in by shader functions
  33. Vertex Shader
  34. Fragment Shader Draw Function
  35. Playback Rate: How fast map plays when you first open it
  36. Master Playback Rate: Scales how fast map plays (how fast 'slow','med','fast' options are)
  37. Colormap Src: For chloropleths?
  38. Layer Description: Description of layer for viewer
  39. Featured Themes
  40. Draw Options: used by Gabriel mostly

URL (column K)

  • Make sure your spreadsheet of data is in a Google Sheets document.
  • While editing that Google Sheets document, go to the share settings and turn link sharing on. Make sure it's set to "Anyone with link can view".
  • Go back to the Google Sheets doc. Copy the link in the search bar and paste it below to generate a new link. This link will work in countries that ban Google products and it will directly download your Google Sheets file as a ".csv"
    New Link:
  • Click on new link to check that it downloads a ".csv" file
  • Paste new link into column k (URL)
